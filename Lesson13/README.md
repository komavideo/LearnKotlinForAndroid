扩展函数
=======

## 知识点

* 类扩展函数

## 实战演习

~~~js
fun String.add(x: Int, y: Int): Int {
    return x + y;
}

fun main(args: Array<String>) {
    var content: String = "Helo Kotlin!";
    println(content);
    println(content.add(10, 20));
}
~~~
